What's the difference between alert() and console.log()?

alert() displays a little popup window with the text inside the parentheses. console.log() displays the text in the parentheses in the console.

What's one situation where you might use an `alert()` to tell the user something?

alert() could be used to initially welcome a user to a page or to tell the user they are being redirected to another page

What's one way that `alert()` might get obnoxious?

Simply if you keep using an alert, then the user will have to click through EVERY single alert box until there aren't any alerts remaining.
